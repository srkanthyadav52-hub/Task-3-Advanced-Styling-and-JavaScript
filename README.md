ApexPlanet Software Pvt Ltd - Internship Task 3
​📋 Project Overview
​This project is a high-performance, responsive web dashboard designed to demonstrate proficiency in modern front-end development. It combines advanced CSS layouts with complex JavaScript logic to create an interactive learning environment and real-time data fetching system.
​✨ Key Features
​1. Adaptive Responsive UI
​Mobile-First Design: Optimized for all screen sizes using CSS Media Queries.
​Grid Architecture: Implemented a dynamic grid system that transitions from a single-column layout (Mobile/Tablet) to a multi-pane dashboard (Desktop).
​Glassmorphism Styling: Utilizes modern UI trends including blurs, gradients, and high-contrast typography for a professional look.
​2. Interactive Technical Assessment
​10-Question Bank: A comprehensive quiz covering core JS and CSS concepts.
​Immediate Response Logic: Instant color-coded feedback (Green for Success / Red for Error) upon selection.
​Dynamic Timer: A 20-second countdown per question to test speed and accuracy.
​Educational Explanations: Real-time technical breakdowns appear after every answer to explain the underlying "why."
​3. Asynchronous API Feed
​Live Fetching: Integrated the fetch() API using async/await syntax to pull live data from a public REST API.
​State Management: Handles loading states and error catching to ensure a smooth user experience even during network interruptions.
​4. Problematic Breakdown & Analysis
​Session Summary: At the end of the session, the system generates a diagnostic report.
​Domain Identification: Categorizes errors into "JS Logic" or "CSS Layout" to identify specific areas for improvement
